Damping Analysis Tool

Compute damping ratio, halving time and decay envelope — full time-series, peak lists, generated signals, or graph images

Study type
○ Free decay — zero baseline
Signal oscillates around zero. Typical for impact hammer tests, free vibration ring-downs. No offset removal needed.
○ Offset oscillation — non-zero baseline
Signal oscillates around a large non-zero mean (e.g. 106 ± 0.5). Baseline is subtracted automatically before analysis.
Baseline is subtracted from every sample before peak detection. All amplitudes in results are relative to this baseline.
Input mode

Two columns: time   value. Accepts comma, tab, semicolon, or space delimiters — mix is fine.

The tool detects peaks automatically from the full signal. If using offset mode, baseline is removed first.

One row per peak: time, amplitude. Amplitudes should already be relative to baseline (offset subtracted manually if needed).
